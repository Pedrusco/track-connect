Quem Somos




A TrackConnect é uma plataforma onde é possível montar uma PlayList integrada. YouTube, Spotify, Deezer, SoundCloud e muito mais em uma única lista de Reprodução.

Como Funciona




É necessario efetuar um cadastro para ter acesso ao nosso sistema. Nele voce poderá incluir as musicas desejadas, podendo ser de diferentes plataformas (YouTube, Spotify, etc), onde ficarão armazenadas dentro de uma tabela.
